T2
Description
一个人拿着刷子刷路，给出要刷的路段，和每次最长能刷的距离，求最短路程
$n\leqslant 500, dStart[i] < dEnd[i] < 2\times 10^9$

Solution
这题倒着贪心，当时想出来了，有个地方写的有问题。
贪心为啥对呢…因为如果刷一次的话，最远路程便是最后一个，如果你还刷了前面的部分，那么第二次刷的路程就会变大。(没怎么想证明…对就vans了)
